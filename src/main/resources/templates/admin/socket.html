⚙️ Socket Architecture (CO4)

            
    
                
  • AlertServer: ServerSocket on port 9090, runs in a
                    daemon thread on startup via @PostConstruct.
    • 
                
  • AlertClient: Socket client connects, sends a command, receives ACK.
                
    • 
                
  • Multithreaded: Each client connection is handled in a separate thread via
                    ExecutorService (CachedThreadPool).
    • 
                
  • Broadcast: All connected clients (regulators) receive the alert simultaneously
                    using CopyOnWriteArrayList<PrintWriter>.
    • 
                
  • Integration: Automatically called by ReportAggregatorService after
                    async report generation completes.
